# 에버타임 (EverTime)

> 에버랜드 대기시간을 실시간으로 확인하고, <br />우리만의 완벽한 동선을 짜기 위해 제작된 모바일 최적화 웹 서비스입니다.

---

## 주요 기능

* **실시간 대기열 확인**: 어트랙션의 현재 대기 시간을 실시간으로 가져와 직관적인 상태 배지(원활/보통/혼잡/운행종료)로 보여줍니다.
* **스마트 검색**: 찾고 싶은 어트랙션 이름을 입력하면 실시간으로 필터링되어 원하는 정보를 빠르게 찾아냅니다.
* **상태별 커스텀 UI**: 검색 결과가 없거나 로딩 중일 때 명확하고 깔끔한 상태 카드를 제공합니다.
* **모바일 최적화**: 현장에서 스마트폰으로 보기에 가장 편하도록 `440px` 모바일 레이아웃에 최적화되어 있습니다.

---

## 기술 스택

* **Framework**: Next.js (App Router)
* **Language**: TypeScript / JavaScript (JSX)
* **Styling**: Pure CSS (Variables, Flexbox 테마 적용)
* **Design Theme**: 에버랜드의 자연을 담은 **Deep Green & Amber** 포인트 컬러 시스템

## 시작하기

### 1. 저장소 클론 및 패키지 설치

```bash
git clone
cd evertime-next
npm install
```

### 2. 개발 서버 실행

```bash
npm run dev
```

서버가 실행되면 브라우저에서 `http://localhost:3000` 접속 후 결과를 확인할 수 있습니다.
